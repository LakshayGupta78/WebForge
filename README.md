# WebForge (VendorHub)

WebForge is the product website and app shell I built to help street vendors go digital.

## What I built

The product combines a landing experience with authentication and a dashboard flow, so vendors can onboard, manage operations, and scale with simple digital tools.

## Core features

- Multi-section landing page for value communication
- Sign up / sign in flow
- Vendor dashboard entry point
- Firebase-backed auth/data layer
- Multi-language support via language context

## Stack

- React (Vite)
- Tailwind CSS
- Framer Motion
- Firebase
- React Router

## Run locally

```bash
npm install
npm run dev
```

## Configuration

Check Firebase setup in `src/lib/firebase.js` and update environment/config values as needed before production deploy.
